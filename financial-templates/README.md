# Financial & Accounting Excel Templates Collection

This directory contains a collection of financial and accounting Excel templates with formulas and VBA macros.

## Template Categories

### 1. Basic Accounting
- **General Ledger** - Track all financial transactions
- **Accounts Payable** - Manage vendor invoices and payments
- **Accounts Receivable** - Track customer invoices and collections
- **Cash Book** - Daily cash transactions recording
- **Bank Reconciliation** - Match bank statements with internal records

### 2. Financial Statements
- **Balance Sheet** - Assets, liabilities, and equity snapshot
- **Income Statement** - Revenue, expenses, and profit/loss
- **Cash Flow Statement** - Operating, investing, financing activities
- **Statement of Changes in Equity** - Equity movements over period

### 3. Budgeting & Forecasting
- **Annual Budget** - 12-month budget planning
- **Departmental Budget** - Budget allocation by department
- **Cash Flow Forecast** - Project future cash positions
- **Sales Forecast** - Revenue projection models

### 4. Financial Analysis
- **Financial Ratios** - Liquidity, profitability, efficiency ratios
- **Break-Even Analysis** - Calculate break-even points
- **Variance Analysis** - Compare actual vs budget
- **Trend Analysis** - Multi-period performance tracking

### 5. Specialized Tools
- **Depreciation Calculator** - Various depreciation methods
- **Loan Amortization** - Payment schedules and interest
- **Payroll Calculator** - Salary, tax, and deductions
- **Invoice Generator** - Automated invoice creation with macros

## Features

- **Pre-built formulas** - All calculations ready to use
- **VBA Macros** - Automation for repetitive tasks
- **Data Validation** - Dropdown lists and input controls
- **Conditional Formatting** - Visual indicators for key metrics
- **Pivot Tables** - Dynamic data analysis
- **Charts & Dashboards** - Visual financial reporting

## How to Use

1. Download the template you need
2. Enable macros when prompted (if required)
3. Enter your data in the input sections
4. Review calculated results
5. Customize as needed for your business

## Security Note

Always review macro code before enabling in untrusted templates. These templates are provided as-is.
